# BlockChain-Cpp-NextGen-Core
新一代区块链底层技术库 C++ 实现，包含隐私计算、Layer2 扩容、分片、预言机、DAG、NFT、MPC 等前沿方向，代码无冗余、无重复、可直接编译运行，适合学习研究与开源展示。

## 项目模块（12大全新功能）
1. 零知识证明验证：区块链隐私交易基础验证逻辑
2. 账户模型系统：以太坊风格账户/转账/余额管理
3. 预言机数据喂价：链下数据上链与去中心化喂价
4. 闪电支付通道：Layer2 链下微支付与扩容方案
5. 分片链路由：高性能区块链分片与地址分配
6. ECDSA 签名验签：BTC/ETH 通用标准签名算法
7. 多方联合签名：MPC 多节点协同签名机制
8. 交易内存池：待打包交易管理与出池调度
9. 区块奖励减半：比特币通缩经济模型实现
10. P2P 消息广播：区块链节点网络消息路由
11. ERC721 NFT 合约：唯一数字资产发行与持有
12. DAG 区块结构：IOTA 类有向无环图账本模型

## 技术亮点
- 纯 C++ 编写，无第三方依赖，跨平台运行
- 覆盖前沿区块链技术：ZKP、MPC、分片、Layer2、DAG
- 代码独立模块化，可直接嵌入项目
- 无重复文件名、无重复逻辑，开源仓库高质量提交

## 运行方式
任意 .cpp 文件使用 g++ 编译即可运行测试。
